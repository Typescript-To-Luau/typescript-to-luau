# Typescript To Luau

A simple transpiler project inspired by [roblox-ts](https://github.com/roblox-ts/roblox-ts), converting Typescript code into **Luau** code.

---

## Getting Started

### Prerequisites
- NodeJS 18+


## Contributing

1. `git clone https://github.com/Typescript-To-Luau/typescript-to-luau`
2. `cd typescript-to-luau`
3. `npm install`
